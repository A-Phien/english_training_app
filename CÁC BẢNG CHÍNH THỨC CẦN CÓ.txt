✅ CÁC BẢNG CHÍNH THỨC CẦN CÓ
1️⃣ users

Quản lý tài khoản

users
- id (PK)
- username (unique)
- email (unique)
- password
- role (ADMIN / USER)
- is_active (boolean)
- created_at
- updated_at

✔ Phục vụ:

Đăng nhập

Phân quyền

Lưu lịch sử luyện tập

2️⃣ lesson

Quản lý video bài học

lesson
- id (PK)
- title
- description
- youtube_url
- video_id
- created_at

✔ Phục vụ:

Admin nhập video

Hiển thị danh sách lesson

Lấy transcript bằng video_id

3️⃣ sentence

Phụ đề được tách thành câu

sentence
- id (PK)
- lesson_id (FK → lesson.id)
- content (TEXT)
- start_time (FLOAT)
- end_time (FLOAT)
- order_index (INT)

✔ Quan hệ:
Lesson 1 — N Sentence

✔ Dùng làm:

Ground truth

So sánh phát âm

4️⃣ evaluation

Lưu kết quả chấm điểm

evaluation
- id (PK)
- user_id (FK → users.id)
- sentence_id (FK → sentence.id)
- transcript (TEXT)
- expected_text_snapshot (TEXT)
- score (DECIMAL)
- mistakes (JSONB)
- created_at

✔ Quan hệ:
User 1 — N Evaluation
Sentence 1 — N Evaluation

✔ Quan trọng:

Snapshot giúp không bị ảnh hưởng nếu sentence bị chỉnh sửa

JSONB giúp lưu lỗi linh hoạt
