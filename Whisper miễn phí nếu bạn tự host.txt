Whisper miễn phí nếu bạn tự host

Whisper là mã nguồn mở: bạn có thể tải model về và chạy nó trên máy của bạn (hoặc server) → không mất tiền API.

📌 Ưu điểm

Miễn phí hoàn toàn cho việc chuyển âm thanh thành chữ.

Không cần API key.

Không tính phút/giây hay token.

❗ Nhược điểm

Cần may đủ mạnh (CPU có thể làm chậm, dùng GPU nhanh hơn).

Phải tự triển khai môi trường (devops / server).

Không thể scale nhiều user dễ như dùng API.

👉 Đây chính là cách miễn phí thực sự cho dev/test/learning

🔹 OpenAI Whisper API

Không có miễn phí vĩnh viễn.

OpenAI thường tặng $5 credits cho tài khoản mới → bạn có thể dùng để test API Whisper và các API khác trong vài tháng.

Sau khi hết credits, bạn phải trả tiền theo số phút audio:

Khoảng $0.003–$0.006 / phút audio âm thanh (tuỳ model, mức này là tham khảo tại thời điểm 2026).

⚠️ Không có “free tier” không giới hạn hoặc miễn phí mãi mãi trong OpenAI Whisper API — dù bạn có API key, nếu hết credit thì bạn trả tiền.
