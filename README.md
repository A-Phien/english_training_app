Hướng dẫn chạy
Thư mục ai-engine:
1:
Trên Windows: .venv\Scripts\activate
Trên Linux/Mac: source .venv/bin/activate
2:
pip install -r requiment.txt
3:
python -m uvicorn main:app --reload --port 8000

Thư mục backend-core:
1: update dependencies trong Maven
2: run Spring boot

Thư mục Frontend-client:
1: npm install
2: npm run dev
